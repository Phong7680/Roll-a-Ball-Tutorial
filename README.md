# Roll-a-Ball Tutorial

## ゲーム概要

**Roll-a-Ball**は、Unityで作成された3Dボールゲームです。プレイヤーはボールを操作し、ステージ内に配置されたアイテムを集めてスコアを獲得します。制限時間内にすべてのアイテムを収集すると勝利となります。

このゲームは、Unity初心者向けのチュートリアルプロジェクトとして設計されており、Rigidbody物理演算、衝突判定、UI表示、コルーチンなどの基本的なゲーム開発要素を学ぶことができます。

## ゲームの目的

- **主な目標**: ステージ内のすべてのPickUpアイテム（12個）を収集する
- **制限時間**: 120秒以内にミッションをクリアする
- **勝利条件**: すべてのアイテムを集めると「You Win!」と表示される
- **敗北条件**: 
  - 制限時間が0になる前に12個集められない場合
  - Enemyに衝突した場合

## 操作方法

### 基本操作
- **移動**: 矢印キー または WASD キー
  - ↑ / W: 前進
  - ↓ / S: 後退
  - ← / A: 左移動
  - → / D: 右移動

### 物理挙動
- ボールはRigidbodyを使用した物理演算で動作します
- カーソルキーの入力に応じて力（AddForce）が加わり、慣性や摩擦の影響を受けます
- 地形や壁に衝突すると、リアルな反発が発生します

## ゲーム要素

### Pick Up（収集アイテム）
- ステージ内に配置された小さな回転するアイテム
- ボールが触れると収集され、スコアが+1される
- 収集したアイテムは消滅します
- 合計12個すべて集めるとゲームクリア

### Enemy（敵）
- ステージ内を移動する敵オブジェクト
- ボールが敵に衝突すると「Game Over!」となり、ゲームが終了します
- 敵を避けながらアイテムを収集する必要があります

### Trap（トラップ）

このゲームには3種類のトラップが配置されており、ボールが触れると特殊な効果が発動します：

#### 1. **Time Trap（時間トラップ）**
- **効果**: 制限時間が10秒減少
- **タグ**: `Time trap`
- **挙動**: 触れた瞬間にトラップは消滅し、タイマーから10秒が引かれる
- **戦略**: できるだけ避けるか、時間に余裕がある時のみ通過する

#### 2. **Stop Trap（停止トラップ）**
- **効果**: ボールが1秒間完全に停止する
- **タグ**: `Stop trap`
- **挙動**: 
  - ボールの速度と回転が0にリセットされる
  - 1秒間、プレイヤーの入力が無効化される
  - 1秒後に操作が再開可能になる
- **戦略**: 敵の近くで停止すると危険なため、安全な場所で触れるように注意

#### 3. **Reverse Trap（逆転トラップ）**
- **効果**: 5秒間、操作が逆転する
- **タグ**: `Reverse trap`
- **挙動**: 
  - 前進キーで後退、左キーで右移動など、すべての入力が反転
  - 効果時間は5秒間
  - 慣れるまで非常に操作が難しくなる
- **戦略**: 効果中は慎重にゆっくり移動し、複雑な操作を避ける

## UI表示

### スコア表示
- **Count**: 現在収集したアイテム数を表示
- リアルタイムで更新されます

### タイマー表示
- **Time**: 残り時間を秒単位で表示
- 0秒になると「Game Over!」

### リザルト表示
- **You Win!**: 12個すべて収集すると表示
- **Game Over!**: 時間切れまたは敵に衝突すると表示

## 技術的な特徴

### 使用技術
- **Unity 2021以降**（Universal Render Pipeline使用）
- **C# スクリプティング**
- **TextMeshPro** によるUI表示
- **新しいInput System** (InputSystem_Actions)
- **Rigidbody物理演算**

### 主要スクリプト
- `PlayerController.cs`: プレイヤーの移動、衝突判定、トラップ処理、スコア管理
- Coroutineを使用した時間制御（Stop TrapとReverse Trapの効果時間管理）

### 物理設定
- Rigidbodyの`AddForce`による力ベースの移動
- Trigger Colliderによるアイテム収集とトラップ検出
- 通常のCollisionによる敵との衝突判定

## プロジェクト構造

```
Assets/
├── Scenes/              # ゲームシーン
├── Scripts/             # C#スクリプト
│   └── PlayerController.cs
├── Materials/           # マテリアル（色、テクスチャ）
├── Pick up/             # 収集アイテムのプレハブ・素材
├── Enemy/               # 敵キャラクターのプレハブ・素材
├── Ball Pack/           # ボールモデル
└── Rolling_Balls-Sci-fi_Pack/  # Sci-fiスタイルのボール素材
```

## ゲームの遊び方

### 方法1: ビルド済みゲームをプレイ（推奨・簡単）

**すぐにプレイしたい方はこちら！**

1. **Gameフォルダをダウンロード**
   - リポジトリから `Game/` フォルダ全体をダウンロードします
   - または [Releases](https://github.com/Phong7680/Roll-a-Ball-Tutorial/releases) から最新版をダウンロード

2. **ゲームを起動**
   - ダウンロードしたフォルダ内の `Roll a Ball Tutorial.exe` をダブルクリック
   - Unityのインストールは不要です

3. **プレイ開始**
   - ゲームが起動したら、矢印キーまたはWASDキーで操作してください

> **注意**: 初回起動時にWindowsセキュリティの警告が表示される場合がありますが、「詳細情報」→「実行」を選択してください。

### 方法2: Unityエディタでプロジェクトを開く（開発者向け）

Unityでプロジェクトを編集したい場合：

1. **プロジェクトを開く**
   - Unity Hub（2021以降推奨）を起動
   - 「Open」→ プロジェクトフォルダを選択
   - Unity Editorでプロジェクトが開きます

2. **ゲームを実行**
   - `Assets/Scenes/` フォルダからメインシーンを開く
   - Unity Editor上部の「▶ Play」ボタンをクリック
   - Gameビューでプレイ開始

3. **ビルドして実行**
   - `File` → `Build Settings`
   - プラットフォームを選択（Windows / macOS / Linux）
   - `Build` をクリックして実行ファイルを生成
   - 生成された実行ファイルを起動
